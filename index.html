2
Configure Chrome DevTools for Agents
Let your agent install, test, and inspect Chrome extensions live
  1. Add the MCP server. Navigate to:
    Settings › Customization › Add MCP server
    Search for Chrome DevTools and add it.
  2. Open the MCP configuration and add these two parameters:
    args 
    --categoryExtensions,
--autoConnect
    The resulting MCP configuration looks like this: 
    {
  "mcpServers": {
    "chrome-devtools": {
      "command": "npx",
      "args": [
        "-y",
        "chrome-devtools-mcp@latest",
        "--categoryExtensions",
        "--autoConnect"
      ]
    }
  }
}
  3. Open Chrome and go to:
    chrome://inspect/#remote-debugging
    Enable Allow remote debugging for this browser instance.
  4. Restart Antigravity to apply all changes.

Then enable remote debugging in Chrome:

  1. Open Chrome and navigate to:
    chrome://inspect/#remote-debugging

Give it a try!

Now that your agent is configured, try building these sample extensions in 4 progressive steps.

Step 1

Hello World

A simple baseline extension to verify that your agent can write, install, and test UI components.

Prompt 
Build a Chrome Extension that opens a popup and displays a styled "Hello World" message.
Step 2

Tab Completion

A more advanced interactive extension that queries active tabs and handles keyboard/text input event injection.

Prompt 
Build a Chrome Extension that offers global `!!tab` autocompletion in text inputs. When typed, display a dropdown overlay of open tabs; selecting one inserts its link into the field.
Step 3

Publish the Extension

See the Chrome Webstore skill in action.

Prompt 
Prepare this Chrome Extension for publication on the Chrome Web Store.
Step 4

Contextual Anchor Text Generator

Build a smart AI-powered utility that leverages Chrome's built-in local Gemini Nano model to rewrite bare links into descriptive anchor texts.

Prompt 
For Markdown links, use Chrome's built-in local AI API to generate a 3-8 word anchor text from the page title and url that matches the existing text input.
